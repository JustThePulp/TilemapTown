Tilemap Town

Collaborative map-building hangout

Controls
  • Movement: Arrow keys/WASD
  • Placing tiles: click on "Inventory" to open it, and click on the tile you want to place.
  • Selecting tiles: click and drag on the map. You can then delete the selected area. Esc will cancel the selection.
Commands

For the full list, see the repository. This section will list common commands.

/me text
Do a pose/action in the area you're at.

/ooc text
Send a message that's explictly marked as being out-of-character..

/spoof text
Narrate something that happened in the area you're at.

/roll dice sides
Do a dice roll.

/nick newname
Change your nickname.

/tell user message
Private message a user. You may click on a private message to reply to it quickly.

/who
List all users on the map.

/gwho
List all users on the server.

/whereare
List public maps that have users on them.

/publicmaps
List all public maps.

/look user
Displays the description of another user on the map, if they have one set.

/userdesc text
Sets your description.

/login username password
/register username password
/changepass password
Manage your account. A registered account is required for some things (like mail), but the main purpose is to save your character information and location for next time.

Setting up an avatar

You can use custom avatars on Tilemap Town! Currently there's no way to upload them to the server so you will need to upload them to an image hosting site; currently the server is set up to accept images from Imgur, ImgBB and Postimages.

For creating the actual image you'll use for your character, there are templates available on GitHub. Player animation frames are 32x32 the bottom center will be aligned with the bottom center of the tile you're standing on. The width and height of the image is used to determine how many animation frames and directions the player graphic has, so pick a template that matches your needs.

Once you have an image uploaded to an allowed image hosting site, use /userpic your_image_url_here. Make sure this is actually a link to the PNG file itself, and not just to a webpage that has the image on it. On Imgur this can mean right clicking on the image and copying its link.
